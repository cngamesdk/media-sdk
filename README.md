# media-sdk
广告媒体SDK。 聚合常见广告媒体，如：腾讯广告、巨量引擎(头条)、磁力引擎(快手)、百度广告、UC、TapTap等，提供了Token获取、请求封装、响应解释等功能，以本地化方式轻松完成API的调用和结果的获取，旨在帮助开发者快速搭建投放管理系统。

## 各媒体入口

- [巨量引擎(头条)](https://open.oceanengine.com/)
- [腾讯广告](https://developers.e.qq.com/)
- [磁力引擎(快手)](https://developers.e.kuaishou.com/welcome)
- [百度广告](https://dev2.baidu.com/home)
